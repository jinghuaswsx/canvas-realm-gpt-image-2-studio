2. 文生图 — text_to_image

POST/images/generations

curl

curl -sS http://172.30.254.14:82/v1/images/generations \
  -H "Authorization: Bearer sk-c742f616059d164c5be23042ab432443023e01ab671db54cff7291bb97676c73" \
  -H "Content-Type: application/json" \
  -d '{
    "model": "gpt-image-2",
    "prompt": "一只在竹林里吃苹果的熊猫，水墨画风格",
    "n": 1,
    "size": "1024x1024"
  }'

Python

import requests

resp = requests.post(
    "http://172.30.254.14:82/v1/images/generations",
    headers={
        "Authorization": "Bearer sk-c742f616059d164c5be23042ab432443023e01ab671db54cff7291bb97676c73",
        "Content-Type": "application/json",
    },
    json={
        "model": "gpt-image-2",
        "prompt": "一只在竹林里吃苹果的熊猫，水墨画风格",
        "n": 1,
        "size": "1024x1024",
    },
)
data = resp.json()
for item in data["data"]:
    if "b64_json" in item:
        import base64
        with open("output.png", "wb") as f:
            f.write(base64.b64decode(item["b64_json"]))
    elif "url" in item:
        print(f"download: {item['url']}")

3. 图生图 / 图片编辑 — image_to_image / edit_image

POST/images/edits

使用 multipart/form-data 上传参考图和提示词。

curl

curl -sS http://172.30.254.14:82/v1/images/edits \
  -H "Authorization: Bearer sk-c742f616059d164c5be23042ab432443023e01ab671db54cff7291bb97676c73" \
  -F "model=gpt-image-2" \
  -F "image=@参考图.png" \
  -F "prompt=把背景换成星空，保持主体不变" \
  -F "n=1" \
  -F "size=1024x1024"

Python

import requests

resp = requests.post(
    "http://172.30.254.14:82/v1/images/edits",
    headers={
        "Authorization": "Bearer sk-c742f616059d164c5be23042ab432443023e01ab671db54cff7291bb97676c73",
    },
    files={"image": open("reference.png", "rb")},
    data={
        "model": "gpt-image-2",
        "prompt": "把背景换成星空，保持主体不变",
        "n": "1",
        "size": "1024x1024",
    },
)

JavaScript / Node.js

const form = new FormData();

Request Body (multipart/form-data)

字段类型必填说明
modelstring / text是gpt-image-2
imagefile是参考图片文件 (png / jpg / webp)
promptstring / text是编辑指令，描述期望的变化
nstring / text否生成数量
sizestring / text否1024x1024 / 1024x1536 / 1536x1024

4. 响应格式

成功 (200)

{
  "data": [
    {
      "b64_json": "iVBORw0KGgoAAAANSUhEUgAA...（Base64 编码的 PNG）",
      "mimeType": "image/png"
    }
  ]
}

两种返回方式：

  • b64_json — 直接在响应里返回 base64 图片数据
  • url — 返回一个下载链接，需要再 GET 下载

处理逻辑（推荐）

data["data"].forEach(item => {
  if (item.b64_json) {
    // 直接解码
    const bytes = Buffer.from(item.b64_json, "base64");
    fs.writeFileSync("output.png", bytes);
  } else if (item.url) {
    // 从 URL 下载
    const response = await fetch(item.url);
    const buffer = await response.arrayBuffer();
    fs.writeFileSync("output.png", new Uint8Array(buffer));
  }
});

5. 参数速查

参数接口类型必填可选值 / 说明
model两者string是gpt-image-2
prompt两者string是中文/英文提示词，建议 <1000 字符
n两者integer/string否1-10，默认 1
size两者string否1024x1024 / 1024x1536 (竖) / 1536x1024 (横)
imageeditsfile是参考图文件，支持 png / jpg / webp

6. 错误码

HTTP 状态含义返回示例
200成功正常 data 数组
400参数错误{"error": {"message": "model is required"}}
401API Key 无效或缺少{"error": {"message": "Unauthorized"}}
429频率限制{"error": {"message": "Rate limit exceeded"}}
500服务器/上游错误{"error": {"message": "Internal server error"}}

7. OpenAI SDK 直接接入

因为是完全的 OpenAI 兼容格式，OpenAI 官方 SDK 可以直接用：

Python

from openai import OpenAI

client = OpenAI(
    api_key="sk-c742f616059d164c5be23042ab432443023e01ab671db54cff7291bb97676c73",
    base_url="http://172.30.254.14:82/v1",
)

response = client.images.generate(
    model="gpt-image-2",
    prompt="一只在竹林里吃苹果的熊猫",
    n=1,
    size="1024x1024",
)

print(response.data[0].url)      # 如果返回 url
print(response.data[0].b64_json)  # 如果返回 base64
